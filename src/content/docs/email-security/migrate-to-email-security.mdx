This page aims at showing you how to perform Area 1 actions in [Zero Trust Email Security](/cloudflare-one/email-security/).

import { Tabs, TabItem } from '~/components';

## Invite users

<Tabs>

  <TabItem label="Area 1">
    1. Go to **Settings** (gear icon) > **Users and Actions**
  </TabItem>
  <TabItem label="Email Security">
    1. Go to **Manage Account**
    2. Select **Members** > **Invite**. Refer to [Add account members](/fundamentals/setup/manage-members/manage/#add-account-members) for detailed instructions.
    3. Assign [roles](/cloudflare-one/roles-permissions/#email-security-roles)
  </TabItem>

</Tabs>

## Contact support

<Tabs>

  <TabItem label="Area 1">
    Send an email to:
    - support@area1security.com
    - phishguard@area1security.com (PhishGuard customers only)
  </TabItem>
  <TabItem label="Email Security">
   Refer to:
    - [Cloudflare Help Center](https://www.support.cloudflare.com/s/?language=en_US)
    - [Add participants to open support tickets](/support/contacting-cloudflare-support/#add-participants-to-open-support-tickets)
    - [Raise a ticket](/support/contacting-cloudflare-support/#getting-help-with-an-issue) by entering the correct answers to the following questions:
        - What type of question do you have?: **Technical - Other Products**
        - In what area can we help you?: **Email Security**
  </TabItem>

</Tabs>

## Search emails

<Tabs>

  <TabItem label="Area 1">
    - Select **Search**:
        1. Fielded search
        2. Freeform search
    - Report false negative/false positive
  </TabItem>
  <TabItem label="Email Security">
   - Select **Email Security** > **Investigation**:
    - Advanced screen
    - Regular screen
    - Popular screen
   - Reclassify messages
  </TabItem>

</Tabs>

## Check metrics

<Tabs>

  <TabItem label="Area 1">
    - Select **Home**
    - **Retractions**
  </TabItem>
  <TabItem label="Email Security">
   - Select **Email Security** > **Monitoring** > **Auto-moves**
  </TabItem>

</Tabs>
